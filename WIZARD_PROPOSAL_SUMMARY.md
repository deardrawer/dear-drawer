# 위자드 에디터 개선 제안서 (요약본)

## 📌 핵심 요약 (Executive Summary)

### 문제
현재 청첩장 에디터는 **4개 탭에 21개 아코디언**으로 구성되어 있어 사용자가 "어디서 뭘 해야 하는지" 혼란스러워합니다.

### 솔루션
**단계별 위자드 플로우**로 전환하여 사용자가 순차적으로 청첩장을 완성하도록 가이드합니다.

### 예상 효과
- **청첩장 완성률**: 35% → 65% (+85% 증가)
- **평균 작성 시간**: 45분 → 30분 (-33% 단축)
- **사용자 만족도**: NPS 60+ 목표

---

## 🎯 위자드 6단계 플로우

```
Step 1: 템플릿 & 기본 정보 (필수)
  └─ 신랑/신부 이름, 결혼 날짜/시간, 예식장 정보

Step 2: 인트로 디자인 (필수)
  └─ 인트로 스타일, 커버 이미지, 카카오 썸네일, 배경음악

Step 3: 인사말 & 스토리 (선택)
  └─ AI 스토리 생성 또는 직접 작성, 커플 프로필, 인터뷰

Step 4: 본문 콘텐츠 (선택)
  └─ 갤러리, 오시는 길, 안내사항, RSVP 설정

Step 5: 마무리 정보 (선택)
  └─ 계좌 정보, 연락처, 감사 인사

Step 6: 미리보기 & 완성
  └─ 슬러그 설정, 공개 설정, 저장 및 공유
```

## 📊 현재 vs 위자드 비교

| 항목 | 현재 (탭 방식) | 위자드 방식 |
|------|----------------|-------------|
| **구조** | 4개 탭, 21개 아코디언 | 6단계 선형 플로우 |
| **진행률 표시** | ❌ 없음 | ✅ 각 단계마다 표시 |
| **다음 할 일** | ❌ 불명확 | ✅ "다음 단계" 버튼 명확 |
| **필수 항목 검증** | ⚠️ 저장 시점 | ✅ 각 단계 완료 시 |
| **AI 추천** | ⚠️ 사용자가 직접 탐색 | ✅ Step 3에서 자동 권장 |
| **모바일 지원** | ❌ 경고 메시지만 | ✅ Step 1-2 최적화 (MVP) |
| **평균 작성 시간** | 45분 | 30분 (목표) |
| **완성률** | 35% | 65% (목표) |

- **Week 6**: 모바일 최적화, 도움말 투어
- **Week 7**: 성능 최적화, A/B 테스트

## 💡 핵심 기능

### 1. 진행률 표시
```
Step 3/6  스토리 만들기
██████████░░ 50% 완료
```

### 2. 단계별 실시간 검증
- 필수 항목 미입력 시 "다음 단계" 버튼 비활성화
- 에러 메시지 즉시 표시 (저장 시점까지 기다리지 않음)

### 3. AI 자동 권장
- Step 3 진입 시 "AI로 만들기" 권장 모달 자동 표시
- 사용자 선택: [AI로 만들기] / [직접 작성하기]

### 4. 모드 전환
- 기존 청첩장 수정 시: "위자드 모드" ↔ "자유 편집 모드" 선택 가능
- 신규 청첩장: 위자드 모드 기본값
